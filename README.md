# student-analysis
<h2> Overview </h2>
<h4>This program helps teachers analyse the progress of their students by using graphical and statistical measures like mean, median, mode, and more. This Program is made using Three Python Modules name Pandas, Matplotlib and numpy.</h4>
<h2> Features </h2>
<h4>This program takes a CSV file and gives : </h4>
<h4>- Mean, Median and Mode</h4>
<h4>- Highest and Lowest Marks</h4>
<h4>- Top 5 Student Names</h4>
<h4>- Marks distribution</h4>
<h4>- Grade distribution</h4>
<h2> technology used</h2>
<h4>- Python</h4>
<h4>- Pandas</h4>
<h4>- NumPy</h4>
<h4>- Matplotlib</h4>
<h4>- CSV File Handling</h4>
<h2>Steps to install & run the project</h2>
<h4>1. Install Python on your system.</h4>
<h4>2. Install the required libraries using the command:</h4>
<h4>   pip install pandas numpy matplotlib</h4>
<h4>3. Place your CSV file (marks.csv) in the same folder as the script.</h4>
<h4>4. Run the Python file using:</h4>
<h4>   python main.py</h4>
<h4>5. View the statistical results in the terminal.</h4>
<h4>6. The graphs will open automatically.</h4>
